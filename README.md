# 🌿 Bot WhatsApp — El Sosiego

Agente de IA para gestionar los mensajes de WhatsApp Business del número 694 268 895.

## Variables de entorno necesarias en Railway

| Variable | Valor |
|---|---|
| `GEMINI_API_KEY` | Tu API key de Google AI Studio |
| `TU_NUMERO` | Tu número personal en formato `34677473362@c.us` |

## Cómo funciona

- **Preguntas simples** → el bot responde automáticamente (precios, menús, ubicación, info general)
- **Preguntas complejas** → el bot manda un "te respondemos en breve" al cliente y te avisa a ti con una sugerencia de respuesta
